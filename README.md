# Cyber security project

This aim of this project is to create a webapp that is insecure by design. The app should contain at least five flaws from the OWASP top ten list.

## Vulnerabilities

These are the vulnerabilities present in the app:

1. Injection
1. Broken authentication
1. Broken access control
1. Cross-site scripting XSS
1. Cross-site request forgery CSRF (Security Misconfiguration)
